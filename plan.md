# 五子棋游戏 - 需求文档

## 1. 游戏功能
1. 棋盘：15×15 标准五子棋棋盘。
2. 落子：鼠标点击落子，黑棋先下，黑白交替。
3. 规则：同一个位置不能重复落子。
4. 胜负判断：横、竖、斜任意方向连成5子即胜利。
5. 胜利后：弹出提示，禁止继续落子。
6. 重新开始：提供按钮，点击清空棋盘，重新游戏。

## 2. 界面要求
1. 棋盘居中显示，大小 600x600px。
2. 棋盘背景为浅木色，线条清晰。
3. 黑棋、白棋明显区分，带边框。
4. 顶部显示当前轮到谁下棋。

## 3. 技术要求
1. 使用原生 HTML + CSS + JavaScript。
2. 不使用任何框架、库。
3. 代码结构清晰，分文件编写。
4. 代码注释详细，适合新手学习。
